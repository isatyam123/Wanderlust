# Wanderlust

Wanderlust is a full-stack travel accommodation platform inspired by modern booking systems.

## Features
- User Authentication
- Property Listings
- Reviews & Ratings
- Cloudinary Image Uploads
- Mapbox Location Search
- MongoDB Atlas Database

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- Passport.js
- Cloudinary
- Mapbox
- EJS
- Bootstrap
